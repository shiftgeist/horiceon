# Shortcuts

## TMUX

- Min-/Maximise current pane: `c-b` + `z`
- Backward Search: `ctrl` + `shift` + `b`
- Resize pane: `c-b ctrl+[arrow-key]`

## Shell

- delete word after cursor: `ctrl` + `w`
- delete word before cursor: `alt` + `d`
- undo: `ctrl` + `_`
- delete from cursor to start: `ctrl` + `u`
