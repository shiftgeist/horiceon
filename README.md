## Installation

**Required** on Arch: `base-devil` `git` `chezmoi`

```bash
chezmoi init git@github.com:shiftgeist/horiceon.git
```

Usefull AUR packages besides yay-core

```bash
yay -S brave-bin spotify notion-app bitwarden-bin typora
```
